# Sound Files

## page-flip.mp3

A soft paper rustle sound (~0.5s) for page flip animation.

**Recommended source:** Free sound effects from:
- https://freesound.org (search "paper flip" or "page turn")
- https://mixkit.co/free-sound-effects/

**Requirements:**
- Duration: ~0.5 seconds
- Format: MP3
- Volume: Should be subtle, not jarring

Place the downloaded file as `page-flip.mp3` in this directory.
